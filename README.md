🍄 Mario Game – Dockerized Node.js App
📌 Overview

This project is a Dockerized Mario Game application built using Node.js. The application is containerized using Docker, making it easy to deploy and run on any system without worrying about dependencies.

Steps:
1) Make an AWS Instance in Ubuntu Machine and add inbound rule as 8080
   
2)Install Docker

3)Clone the REpositories

4)Make Dockerfile

5)Build the image and expose port map

6)Paste public IP in your browser with port 8080
